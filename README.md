# Convert-Word-Libre-Office-files-to-HTML-with-simple-CSS
This Python script converts Microsoft Word (.doc, .docx) , LibreOffice (.odt)  HTML files into a minimal, clean HTML/CSS package. I  Features     • Converts .doc, .docx, .odt to HTML using LibreOffice (headless).     • Accepts existing .html / .htm files directly.     • Produces a portable folder containing .html, .css, and images/.
